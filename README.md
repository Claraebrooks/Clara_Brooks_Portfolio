# Clara Brooks Portfolio Website

This is an editable static portfolio website built with HTML, CSS, and JavaScript. It can be hosted free on GitHub Pages, Netlify, or Vercel and linked directly on a resume.

## Files

- `index.html` — all page content
- `styles.css` — colors, layout, typography, and responsive design
- `script.js` — mobile menu and marketing portfolio filters
- `assets/` — place your resume PDF and project images here

## First edits to make

Open `index.html` in a text editor and replace:

- `YOUR-EMAIL@example.com`
- `YOUR-LINKEDIN`
- `YOUR-GITHUB`
- `YOUR-SHARKID-REPO`
- Expected graduation year
- GPA, coursework, dates, job titles, and specific accomplishments

Search the file for the words `Add`, `Optional`, and `placeholder` to find editable areas.

## Add your resume

1. Export your resume as a PDF.
2. Name it exactly `Clara-Brooks-Resume.pdf`.
3. Put it inside the `assets` folder.

## Add images

Add image files to the `assets` folder. Replace a placeholder block such as:

```html
<div class="marketing-image placeholder-image">Add campaign image</div>
```

with:

```html
<img class="marketing-image" src="assets/your-image.jpg" alt="Description of the project image">
```

## Preview locally

Double-click `index.html` to open it in a browser. For the best preview, use the VS Code Live Server extension.

## Publish with GitHub Pages

1. Create a new public GitHub repository, such as `portfolio`.
2. Upload all files from this folder.
3. In the repository, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder, then save.
6. GitHub will generate a public website link you can place on your resume.

A custom domain can be added later.
