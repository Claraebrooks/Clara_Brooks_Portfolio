Independent Project • In Development

SharkID

SharkID is a proposed web application that uses machine learning to identify shark species from uploaded photos. It is designed to display species information, conservation status, habitat, diet, size, tooth imagery, and a GIS-based distribution map.

For species with unique markings, such as whale, zebra, and leopard sharks, a future version could match repeat sightings to help track individual animals through community-submitted images.
